Programação

Horário Palestra
08h30
Abertura
09h00

Não existe desculpa!

John Murowaniecki - Encode Web

Sublime Text, gEdit, TextWrangler, Geany, Vim, entre outros aplicativos possúem N+1 plugins e recursos de desenvolvimento que são - na melhor das hipóteses - ignorados. Não existe desculpa pra entregar código ilegível, feio ou simplesmente ruim. Utilizaremos como exemplo o hype Sublime Text e o cult (e amplamente disponível) Vim.
9h40

Comunicação P2P arbitrária entre browsers

Guilherme Vieira - E-Core

Bastante se é falado sobre o padrão WebRTC, que permite comunicação P2P entre browsers, mas o foco geralmente é em streaming de áudio e vídeo, sendo que é possível enviar qualquer tipo de dado. Vamos explorar uma pequena aplicação que sincroniza playback de vídeo entre dois browsers, permitindo que duas pessoas vejam um filme "juntas" pela Internet com facilidade.
10h20
Lanche
10h40

Javascript no servidor com Node.js

Paulo Diovani Gonçalves - Codeminer42

Uma introdução a Node.js, apresentando as principais características (e vantagens) da plataforma, seu ecossistema e comunidade crescente. Ao final, apresenta algumas dicas de como aprender Node.js rapidamente.
11:20

Pinky Promise

Diogo Pereira Ribeiro - Hewlett-Packard

O que são promises, que problema elas resolvem e como utilizá-las de maneira efetiva em aplicações reais. Cobrirei alguns casos de uso mais complexos, tratamento de exceções disparadas assincronamente e algumas comparações breves com outras estruturas de controle de execução de tarefas assíncronas utilizadas em outras linguagens.
12h00
Rango
14h00

Flash talk: Firefox OS e o mobile de amanhã

Cynthia Zanoni - Think Inovação

Tu já conheces o Firefox OS? E se eu te contasse que tu podes contribuir para o futuro dele? Conheça as novidades da plataforma mobile da Mozilla e saiba como fazer parte desta história.
14h20

#2 cases interativos com JS

Luiz Sordi - Huia

No primeiro, trataremos da utilização de webcam para detecção de gestures e reconhecimento facial. Já no segundo projeto, mostraremos o uso de 3D (WebGL) com realidade virtual, além de integração com Oculus Rift. Os cases foram desenvolvidos na Huia e serão explorados desde o problema até a solução técnica.
15h00

Ciência de foguete, usando reactJS para criar a futura plataforma de vídeos da globo.com

Túlio Ornelas - Globo.com

Uma troca de experiência sobre o desenvolvimento de um grande projeto em ReactJS, explorando os desafios e as soluções encontradas. A internet esta cheia de exemplos com um palmo de profundidade veja como as coisas são feitas de verdade.
15h40
Lanche
16h00

O JavaScript do futuro, hoje e agora, serião mesmo.

Fabrício Matté - Nodo

Apresentarei uma forma eficiente e prática de trabalhar a sério com as novas e futuras features da linguagem JavaScript. Esta palestra tem o foco em otimizar seu workflow de desenvolvimento com ECMAScript.next, através de builds incrementais, testes unitários e afins para que você possa desenvolver da forma mais agradável.
16h40

Criação de APIs lightweight com harvester.js

Diogo Lucas - AGCO

Tecnologias como node.js, MongoDB e ElasticSearch são um fit fantástico para a criação de APIs leves e escaláveis. Nesta apresentação, conheceremos o harvester.js, uma biblioteca de apoio na criação dessas APIs, focada na eliminação de código desnecessário e no uso de padrões abertos.
17h20

Processamento Digital de Sinais com Javascript

Rafael Specht da Silva - Grupo RBS

As novas APIs da web tem possibilitado grandes feitos para o Javascript. Essa palestra aborda tratamento de sinais que vão desde a criação de filtros de imagem, passando por distorções de áudio (e pedais de guitarra!), até compressão de streams! Vamos falar sobre as barreiras que a linguagem quebra diariamente!
18h00
